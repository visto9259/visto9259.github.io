Zendframework MVC Webpack Integration

This package work in tandem with php-webpack-plugin

Requirements

Please see the composer.json file for dependencies.

Installlation

Using Composer:

$ composer require visto9259/zend-webpack

Or, alternatively, manually add the following to your composer.json, in the require section:

{
  "require": {
    "visto9259/zend-webpack" : "^1.0.0"
  }
}
